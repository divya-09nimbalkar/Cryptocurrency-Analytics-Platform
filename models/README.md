# Cryptocurrency Analytics Platform

A modular Python project for analyzing and visualizing cryptocurrency datasets.  
Built with **pandas**, **matplotlib**, and a clean project structure.

---

## 🚀 Features
- Load raw cryptocurrency data from CSV
- Perform basic analytics (price, volume, summary stats)
- Visualize results with charts
- Organized structure for scalability
